## Project Overview

This project is a simple single-page professional portfolio website developed using semantic HTML5.

The website presents my professional background, education, technical skills, work experience and selected software-development projects.

## Running the Project

1. Open the project folder in Visual Studio Code.
2. Open the `index.html` file.
3. Run it using Live Server or open it directly in a web browser.
4. Use the navigation links to move between the sections.

## HTML Validation

The `index.html` file should be tested using the W3C HTML Validator before submission.

Validation helps identify:

* Missing closing tags
* Incorrect HTML nesting
* Duplicate IDs
* Missing image alternative text
* Invalid HTML attributes
